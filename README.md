# SMART-Resume-analyzer
The Smart Resume Analyser is an intelligent web application designed to help job seekers optimize their resumes for modern hiring systems.
Most large companies use automated software called Applicant Tracking Systems (ATS) to screen applications before a human recruiter ever sees them. If a resume is missing the right keywords or uses complex layouts like tables and columns, it can be filtered out automatically.

This tool solves that problem by bridge-building between your profile and your dream job. When a user uploads their resume alongside a target job description, the platform analyzes the text in real time. It screens the document for crucial hard skills, software tools, and professional competencies required by the employer.

Key Features Include:
ATS Compatibility Scoring: Evaluates document formatting, fonts, and structure to ensure it can be parsed flawlessly by automated hiring software.

Keyword Matching Engine: Scans for exact phrases and industry-specific terms to see how well the resume aligns with the job description.

Weak Areas Detector (GapLens): Performs a negative lookup to pinpoint critical missing skills, framework deficiencies, or experience gaps. It provides actionable recommendations on exactly what keywords or projects to add to increase compliance.

By transforming the resume review process into data-driven feedback, this analyser helps candidates fix hidden formatting flaws, fill crucial skill gaps, and significantly increase their chances of landing an interview.
